# 🏋️ Academia - Landing Page

Este repositório contém a interface principal de um sistema de gestão para academias, desenvolvido com foco em performance e experiência do usuário (UX).

## 🚀 Sobre o Projeto
O projeto foi idealizado para ser um ecossistema completo, dividindo a experiência entre **Gestores** (Administrativo) e **Alunos** (PWA/Mobile). No momento, este repositório exibe a **Landing Page** de conversão.

### ✨ Funcionalidades em Desenvolvimento:
- **Painel do Aluno (PWA):** Acesso a treinos, evolução física e financeiro.
- **Gestão Administrativa:** Controle de planos, matrículas e estoque da loja.
- **Loja Integrada:** Venda de suplementos e acessórios diretamente pelo app.
- **Login Multi-nível:** Separação segura entre acessos de clientes e usuários.

## 🛠️ Tecnologias Utilizadas
- **PHP 8.x** (Lógica de servidor)
- **MySQL** (Persistência de dados)
- **Arquitetura PWA** (Progressive Web App para mobile)
- **Bootstrap/CSS3** (Interface Responsiva)

## 📂 Estrutura Exposta
Atualmente, o repositório foca na entrega da página inicial:
- `index.php`: Página principal com foco em conversão e planos.

> **Nota:** Os módulos internos e lógicas de backend sensíveis estão em ambiente privado para garantir a segurança da regra de negócio.
